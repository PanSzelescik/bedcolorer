# colorize
Right click with the dye on some blocks to color them!

### Supported Blocks:

- Banner
- Bed
- Candle
- Carpet
- Concrete
- Concrete Powder
- Shulker Box
- Stained Glass Block
- Stained Glass Pane
- Terracotta
- Terracotta Glazed
- Wool
